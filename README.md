# BAGEL2

BAGEL2 software

The Bayesian Analysis of Gene Essentiality 2 (BAGEL2) is a robust gene essentiality identifier from CRISPR-cas9 genome wide pooled library screen data.

## Download BAGEL2

```
git clone https://github.com/hart-lab/bagel
```
