# BAGEL2

BAGEL2 software

The Bayesian Analysis of Gene Essentiality 2 (BAGEL2) is a robust algorithm for accurate measurement of essentiality of genes from single hairpin RNA and CRISPR-cas9 genome wide genetic screens.
